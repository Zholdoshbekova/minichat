How to run the project locally
1. clone the repository
   git bash
   git clone <your repository url>

2. Navigate to the project
   git bash
   cd minichat
3.Install all necessary dependencies
  git bash
  npm install
4. Start  the project in development node
   git bash
   npm run dev
